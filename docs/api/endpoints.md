# API Endpoints

## Health Check
- URL: `/health`
- Método: GET
- Respuesta: `{"status": "healthy", "timestamp": ...}`

## API Status  
- URL: `/api/status`
- Método: GET
- Respuesta: `{"api_status": "available", ...}`

## System Info
- URL: `/info`
- Método: GET
- Respuesta: Información detallada del sistema
